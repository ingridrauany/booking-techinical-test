### Executando a aplicação

Primeiro baixe os pacotes da aplicação:

```bash
npm install
```

Depois execute o servidor de desenvolvimento:

```bash
npm run dev
```

Abra [http://localhost:5138](http://localhost:5138) com seu navegador para ver o resultado.
